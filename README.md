# Pokémon UI Clone – Assignment 1

This project is my first frontend UI assignment built using HTML and CSS. In this assignment, I recreated a Pokémon-themed landing page inspired by a modern UI design. The main purpose of this project was to practice and improve my understanding of CSS positioning concepts and real-world layout creation.

While building this project, I learned how to use different CSS properties such as `position: relative`, `position: absolute`, Flexbox, border-radius, spacing, alignment, and image positioning. I also explored how layered UI elements work together to create attractive website designs. The project includes a split-screen layout, custom positioned elements, a Pokémon character image, navigation dots, a menu icon, and a Pokéball created using CSS.

One of the biggest learnings from this assignment was understanding how parent and child positioning works in real UI layouts. I also learned how to structure a webpage properly and make the design visually balanced.

### Technologies Used

* HTML5
* CSS3
* Remix Icons
